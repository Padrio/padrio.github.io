## About the Project

Toolstone Privacy Manager is a tool developed in C# that helps users limit data collection and tracking in Windows 10/11. The tool offers a user-friendly interface for managing privacy settings.

## Technologies

- **Language**: C#
- **Framework**: .NET
- **Platform**: Windows 10/11

## Features

- Easy management of Windows privacy settings
- Support for Windows 10 and 11
- User-friendly GUI
- Comprehensive control over data collection
- Fully reversible changes with just a few clicks

## Media Coverage

The project was featured in articles by renowned German IT magazines including CHIP and Computerbild, both in print and online editions.

## Challenges

The biggest challenge was building a comprehensive list of Windows tracking and telemetry features while developing sustainable solutions to disable them. Each privacy setting needed to be carefully researched to ensure maximum privacy protection while remaining fully reversible with just a few clicks - allowing users to restore default settings at any time without breaking system functionality.
