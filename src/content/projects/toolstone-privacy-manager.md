## Über das Projekt

Toolstone Privacy Manager ist ein in C# entwickeltes Tool, das Benutzern dabei hilft, die Datenerfassung und das Tracking in Windows 10/11 zu begrenzen. Das Tool bietet eine benutzerfreundliche Oberfläche zur Verwaltung von Privacy-Einstellungen.

## Technologien

- **Sprache**: C#
- **Framework**: .NET
- **Plattform**: Windows 10/11

## Features

- Einfache Verwaltung von Windows Privacy-Einstellungen
- Unterstützung für Windows 10 und 11
- Benutzerfreundliche GUI
- Umfassende Kontrolle über Datenerfassung

## Herausforderungen

Die größte Herausforderung war die Integration mit den Windows-Systemeinstellungen und die Gewährleistung der Kompatibilität mit verschiedenen Windows-Versionen.
