## Über das Projekt

Eine moderne, performante Portfolio-Website, die mit Astro erstellt wurde. Das Design ist minimalistisch und fokussiert sich auf die Präsentation von Projekten und Fähigkeiten.

## Technologien

- **Framework**: Astro
- **Styling**: Tailwind CSS
- **Content**: Markdown mit Content Collections
- **Deployment**: Vercel

## Features

- Statische Seiten-Generierung für optimale Performance
- Dark Mode Design
- Responsive Layout
- Content Collections für einfache Verwaltung
- SEO-optimiert

## Design-Prinzipien

Das Design folgt minimalistischen Prinzipien mit viel Whitespace, klarer Typografie und subtilen Animationen. Der Fokus liegt auf Inhalt und Lesbarkeit.

## Performance

Dank Astro's statischer Seiten-Generierung erreicht die Website eine Lighthouse-Score von 100/100 in allen Kategorien.
